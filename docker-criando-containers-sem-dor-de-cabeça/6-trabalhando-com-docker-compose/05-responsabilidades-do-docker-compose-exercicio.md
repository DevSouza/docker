# 05 Responsabilidades do Docker Compose - Exercicio

Quais das descrições podem ser responsabilidades do Docker Compose?

- [x] A) Executar o build de vários containers.
> Correto!

- [x] B) Desligar os containers de maneira coordenada.
> Correto!

- [ ] C) Manipular apenas um container.
> Errado, é possível gerencial vários containers, até mesmo num único arquivo de configuração.

- [x] D) Criar nova rede em qual os containers podem participar.
> Correto, o docker compose cria uma rede padrão. Também é possível criar uma nova rede usando o comando ```docker network```.
